# Zudio Sales and Stock Analysis Dashboard

## Project Overview
This project analyzes Zudio retail sales and stock data to understand product performance, sales trends, and inventory levels. The dashboard provides insights into sales revenue, stock availability, and product demand to help improve inventory management and sales performance.

## Objective
The main objective of this project is to analyze sales and stock information to identify top-selling products, monitor stock levels, and understand sales patterns across different regions and time periods.

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Data Analysis

## Dataset
The dataset includes the following information:
- Product Name
- Category
- Store / Region
- Units Sold
- Sales Revenue
- Available Stock
- Stock Status
- Date of Sale

## Dashboard Features
The dashboard includes the following visualizations:

- Total Sales KPI
- Total Units Sold
- Available Stock Overview
- Sales by Product Category
- Sales by Region
- Monthly Sales Trend
- Top Selling Products
- Low Stock Products Identification

## Key Insights
- Identified best-performing products based on sales.
- Analyzed stock levels to detect low inventory items.
- Compared sales performance across different regions.
- Tracked monthly sales trends to identify peak sales periods.
